## 🛠️ 技术细节

### 场景生成器 (`scene_generator.py`)
- **库:** PIL/Pillow
- **功能:** 程序化生成像素风格场景
- **支持:** 3 个时段变体（白天/黄昏/夜晚）
- **可扩展:** 色板、建筑位置、树木位置可定制

### UI 生成器 (`ui_generator.py`)
- **库:** PIL/Pillow
- **功能:** 生成 UI 元素（对话框/按钮/图标等）
- **特点:** 
  - 圆角矩形绘制函数
  - 三态按钮
  - 数学公式绘制心形
  - 统一暮色村主题色板

### 问题解决
| 问题 | 解决方案 |
|------|----------|
| PIL 模块未安装 | `apt-get install python3-pil` |
| pip 外部管理限制 | 使用 apt 安装而非 pip |
