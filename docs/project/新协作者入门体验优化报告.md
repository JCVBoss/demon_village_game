## 🎯 新协作者指南核心内容

### 5 步快速开始
1. **了解项目**（5 分钟）— 阅读 README.md
2. **找到角色**（3 分钟）— 查看 CONTRIBUTORS.md
3. **领取任务**（5 分钟）— Project Board 领取
4. **开始工作** — 按角色提供具体指引
   - 💻 程序员：Git 工作流、提交规范
   - 🎨 美术：资源规范、提交示例
   - 📝 文档：Tag 格式、文档规范
5. **创建 PR + 写日志** — 完整流程

### 常见问题解答
- Q: 我是新来的，应该用哪个日志目录？
- Q: Co-authored-by 必须写吗？
- Q: 如何知道该做什么任务？
- Q: 遇到问题怎么办？
- Q: 我的贡献会被记录吗？

### 按需阅读指引
- P0 必读（3 个文档）
- P1 按角色（程序员/美术/文档）
- P2 参考资料（backlog、kanban 等）
