# 🏰 魔王城下的最后村庄 - 故事背景与主线

> "有时候，最勇敢的事不是打败魔王，而是保护你想保护的人。"

---

## 🌍 世界观设定

### 时代背景

**"黑潮纪元"第 47 年**

魔王"暗影之主"苏醒已 47 年，人类王国节节败退。曾经繁华的大陆如今只剩零星抵抗，魔王城如同黑色太阳般矗立在大陆中心，吞噬着一切光明。

王国军队组织了七次北伐，七次惨败。勇者传说成了遥远的童话——上一个被称为"勇者"的人，死在魔王城百里之外，尸体被钉在城门上示众。

人们不再相信勇者，不再相信希望。他们只相信一件事：**活下去**。

### 地理位置

**暮色村**位于大陆中部偏北，是魔王城脚下的最后一个人类村庄。

- **北方：** 魔王城方向，黑潮侵蚀区，不可通行
- **南方：** 迷雾森林，通往村庄的唯一出口
- **东方/西方：** 世界边界，不可探索

### 魔王军的真相

魔王并非天生邪恶。47 年前，他是王国最杰出的大魔法师，因揭露王室腐败而被陷害，被迫堕入黑暗。他的"魔王军"中，有许多是被王国抛弃的人——逃兵、流亡者、被放逐的贵族。

**魔王的目的：** 不是毁灭人类，而是推翻腐朽的王国，建立新秩序。

**魔王的困境：** 力量失控，逐渐被黑暗侵蚀。他知道自己在变成真正的"怪物"，但已无法回头。

---

### 勇者的传说

古老的预言说：

> "当黑暗吞噬最后的光明，真正的勇者将从灰烬中诞生。
> 不是手持圣剑的战士，而是理解痛苦的人。
> 不是打败魔王的英雄，而是能做出选择的人。"

没人知道预言的后半句——"能做出选择的人"，要选择什么。

---

## 📖 主线故事

### 序章：勇者的坠落

**"你"（玩家）** 是被王国选中的"勇者"。

没有显赫的身世，没有传奇的经历——只是一个普通的冒险者，被宫廷祭司选中，被告知"你是预言中的勇者"。

你接受了使命，踏上了征途。一路战斗，一路成长，招募伙伴，收集装备。你相信自己在做正确的事。

直到你抵达魔王城脚下的最后一个村庄——**暮色村**。

那一夜，你做了个梦。梦里，一个声音问你：

> "你真的相信，杀死魔王就能拯救世界吗？"

醒来后，你发现所有装备消失了。伙伴不见了。记忆变得模糊。

你只记得一件事：**必须去魔王城**。

但现在的你，手无寸铁。

---

### 第一章：暮色村的十日

失去装备的勇者，被迫在暮色村停留。

村庄看似平静，但你能感觉到暗流涌动。村民对你态度各异——有人欢迎，有人警惕，有人...在观察你。

**十日时限：** 魔王军将在十日后抵达暮色村。

你必须在这十天内：
1. 找回装备（或找到替代方案）
2. 说服村民帮助你（或独自面对）
3. 做出选择——继续"勇者之路"，还是寻找第三条路？

---

### 第二章：秘密的揭露

随着你与村民的互动加深，秘密逐渐浮出水面：

- **陈默** 发现你就是当年战役的"目击者"（你曾远远看到那场战斗）
- **夜鸦** 收到魔王军的进攻指令，陷入两难
- **老约翰** 告诉你预言的真相——"选择"指的是什么
- **影** 揭露王国的腐败，魔王并非真正的敌人
- **小安** 的身世被揭开，陈默必须面对自己的过去

**关键抉择点：**
1. 是否告诉陈默"你就是当年那场战役的目击者"？
2. 是否帮助夜鸦传递假情报给魔王军？
3. 是否相信影的话，质疑王国的正义性？

---

### 第三章：战火逼近

第十日，魔王军抵达。

夜鸦必须做出选择——执行命令，还是背叛？
陈默必须做出选择——继续逃避，还是站出来？
村民必须做出选择——抵抗，还是投降？
你必须做出选择——相信预言，还是相信自己？

## 🎭 角色故事线

### 陈默线：逃兵的救赎

**核心冲突：** "我不配活着" vs "我活着是为了守护"

**关键节点：**
1. 初期：回避过去，只说自己是"万能工"
2. 中期：勇者发现他的秘密，他选择坦白
3. 后期：小安身世暴露，他必须保护孩子
4. 结局：选择战斗，完成救赎

**专属结局条件：**
- 陈默存活 + 小安存活 + 夜鸦倒戈 = 【救赎结局】
- 陈默牺牲 + 小安存活 = 【牺牲结局】
- 陈默逃跑 + 小安被俘 = 【堕落结局】

---

### 夜鸦线：卧底的抉择

**核心冲突：** "任务" vs "感情"

**关键节点：**
1. 初期：冷静观察，传递情报
2. 中期：与村民互动增多，内心动摇
3. 后期：收到进攻指令，必须选择
4. 结局：倒戈/背叛/逃亡

**专属结局条件：**
- 夜鸦倒戈 + 魔王军撤退 = 【光明结局】
- 夜鸦逃亡 + 村民幸存 = 【自由结局】
- 夜鸦执行命令 = 【毁灭结局】

---

### 勇者线：真正的勇者

**核心冲突：** "被赋予的使命" vs "自己的选择"

**关键节点：**
1. 初期：相信自己是"预言中的勇者"
2. 中期：发现预言的真相，开始质疑
3. 后期：理解"选择"的含义
4. 结局：做出自己的选择

**专属结局条件：**
- 选择与魔王对话 = 【和解结局】
- 选择击败魔王 = 【传统结局】
- 选择推翻王国 = 【革命结局】

---

### 小安线：孩子的视角

**核心冲突：** "天真" vs "真相"

**关键节点：**
1. 初期：不知道父亲已死，以为"去了很远的地方"
2. 中期：开始怀疑，询问陈默
3. 后期：得知真相，必须接受
4. 结局：理解父亲的选择，成长

**功能：** 情感核心，触发其他角色的成长

## 🎯 多结局系统

### 结局分支逻辑

结局由三个关键选择决定：

1. **陈默的选择**（战斗/逃跑/牺牲）
2. **夜鸦的选择**（倒戈/执行命令/逃亡）
3. **勇者的选择**（击败魔王/对话/推翻王国）

### 主要结局（8 种）

| 编号 | 结局名 | 条件 | 描述 |
|------|--------|------|------|
| A | 【救赎】 | 陈默战斗 + 夜鸦倒戈 + 勇者对话 | 陈默完成救赎，夜鸦加入反抗军，勇者与魔王达成和解，建立新秩序 |
| B | 【牺牲】 | 陈默牺牲 + 夜鸦倒戈 + 勇者击败魔王 | 陈默为保护小安牺牲，夜鸦幸存，勇者成为真正的英雄但失去朋友 |
| C | 【革命】 | 陈默战斗 + 夜鸦倒戈 + 勇者推翻王国 | 三方联手推翻腐朽王国，建立新政权，陈默成为新军领袖 |
| D | 【毁灭】 | 陈默逃跑 + 夜鸦执行命令 + 勇者失败 | 村子被毁，所有人死亡（坏结局） |
| E | 【自由】 | 陈默逃跑 + 夜鸦逃亡 + 勇者离开 | 夜鸦带部分村民逃离，陈默继续流浪，勇者放弃使命（中立结局） |
| F | 【真相】 | 陈默战斗 + 夜鸦倒戈 + 勇者发现预言真相 | 勇者理解预言真意，选择第三条路，魔王恢复人性 |
| G | 【传承】 | 陈默牺牲 + 夜鸦倒戈 + 勇者培养小安 | 陈默牺牲，小安被勇者培养成新一代"勇者"（传承结局） |
| H | 【轮回】 | 特定隐藏条件 | 揭示这是一个循环，勇者其实已经经历过多次（真·隐藏结局） |

## 🎮 剧情推进机制

### 信任系统

每个村民有隐藏的"信任值"，通过对话和选择影响：

- **+信任**：真诚对话、帮助村民、遵守承诺
- **-信任**：欺骗、强迫、背叛

信任值影响：
- 村民是否愿意帮助你
- 是否解锁隐藏对话
- 结局分支条件

### 情报系统

村民之间会传播信息：

- 你对 A 做的事，B 可能会听说
- 某些秘密会被"意外"揭露
- 可以选择"主动坦白"或"被动揭露"

### 时限系统

十日倒计时：

- 每日可行动 3-4 次（探索/对话/休息）
- 某些事件只在特定日期触发
- 第十日强制进入最终章
