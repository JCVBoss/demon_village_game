# 🏰 魔王城下的最后村庄 - 项目计划

## 📋 项目概述

**核心概念：** 勇者斗魔王的框架，但勇者到了魔王城脚下的最后一个村子时失去了所有装备。需要在约 10 人的村子里寻找装备和伙伴，通过互动说服或打动村民来帮忙。

**核心创新：** 🤖 **双模式对话系统** — 一周目采用精心设计的分支对话保证叙事体验，二周目解锁 LLM 智能体模式，NPC 根据人设自由对话，每次游玩体验完全不同！

---

## 🎮 核心设计（已确认）

### 视觉与布局
| 项目 | 设定 |
|------|------|
| 视觉风格 | 像素风界面 |
| 视角 | 2D 横版（类似星露谷物语） |
| 村庄规模 | 中等偏小，紧凑布局 |
| 建筑数量 | 5-6 栋可进入 + 1-2 公共区域 + 2-3 废弃/上锁 |
| 游戏时长 | 目标约 3 小时 |

### 玩法循环
- 🎨 像素风界面
- 🔍 探索收集（装备 + 对话道具）
- ⚔️ 对话驱动战斗（可触发/避免，战斗中可对话）
- 🎭 信任系统（每个村民有隐藏信任值）

### 战斗系统
- ⚔️ 回合制
- 🎭 发言系统：双模式
  - **一周目**：预设分支对话 + 关键选项引导（保证叙事节奏）
  - **二周目**：解锁 LLM 自由生成模式（根据人设动态回应）
- 📊 属性阈值：战意/信任/恐惧/理智
- 🤝 队友协助：村民连携攻击/援助
- 🤖 LLM 驱动：仅在二周目启用，根据人设和听到的发言动态生成回应

### 技术选型
- 🎮 引擎：Godot（2D 支持好，开源免费，适合像素风）
- 🤖 LLM：云端 API（二周目启用，可考虑本地缓存优化成本）
- 💾 对话系统：一周目使用预设对话树（JSON 配置），二周目切换至 LLM API
- 👥 开发：单人 + 多 Agent 协作模式
- 🌐 展示网页：本地部署（12345 端口），独立于魔王城 git 仓库

## 📝 设定进展

### 角色设定 ✅（10/10）

1. ✅ **陈默（逃兵）** — 退休冒险者/临时守卫
   - 全队最小年龄的兵，被全队掩护突围
   - 保护队长的孩子（小安，教父身份）
   - 核心冲突：从逃避到直面，完成自我救赎

2. ✅ **雷叔** — 断臂的老匠人
   - 前王国军械师，因伤退役
   - 认识陈默的队长，曾为小队打造装备
   - 义肢里藏着短剑

3. ✅ **金铃** — 精明的女商人
   - 前盗贼团"夜鸦"二把手，金盆洗手
   - 知道观察者的真实身份
   - 消息灵通，情报贩卖

4. ✅ **白芷** — 温柔的女医师
   - 被流放的贵族小姐，自学医术
   - 爱人是战死的平民医生（与陈默队长同一战役）
   - 认识队长夫妇，知道牺牲细节

5. ✅ **老约翰** — 神秘的祭司
   - 前王国宫廷祭司，知晓预言
   - 知道勇者失忆/失装备的真相
   - 引导者角色

6. ✅ **大熊** — 豪爽的酒馆主人
   - 退役佣兵团长，团灭唯一幸存者
   - 和陈默队长是旧识
   - 信息中心，村民关系网枢纽

7. ✅ **影** — 独居的怪人
   - 前王国情报官，被追杀隐居
   - 知道魔王军真实目的
   - 利用观察者传递假情报

8. ✅ **小安** — 队长的遗孤（8 岁）
   - 陈默队长的亲生儿子
   - 由陈默暗中保护（不知其真实身份）
   - 情感核心，触发陈默成长线

9. ✅ **阿虎** — 年轻的村守卫
   - 本地青年，第一次承担重任
   - 暗恋金铃
   - 战斗教学，后期可成盟友

10. ✅ **夜鸦** — 魔王卧底
    - 魔王军情报官，潜伏 10 年
    - 对村民产生感情，内心挣扎
    - 剧情反转关键，多结局分支点
