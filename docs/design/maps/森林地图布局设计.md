## 🌿 区域详细设计

### 1️⃣ 入口区（安全区）

| 属性 | 数值 |
|------|------|
| 位置 | 北侧（靠近村庄） |
| 尺寸 | 15×10 瓦片 |
| 植被 | 稀疏树木、草地 |
| 敌人 | 无（安全区） |
| 功能 | 采集草药、教学区域 |

**特征：**
- 光线充足，视野开阔
- 小径清晰，易于通行
- 有少量草药采集点
- 村民偶尔会来这里采集

---

### 2️⃣ 深林区（探索区）

| 属性 | 数值 |
|------|------|
| 位置 | 中部和西侧 |
| 尺寸 | 20×15 瓦片 |
| 植被 | 密集树木、蘑菇圈、古老树林 |
| 敌人 | 低阶魔物（侦察兵） |
| 功能 | 探索、战斗、宝箱 |

**特征点：**

| 地点 | 位置 | 说明 |
|------|------|------|
| 蘑菇圈 | (10, 5) | 发光蘑菇，可采集 |
| 古老树林 | (25, 5) | 3 棵巨大古树，有秘密 |
| 小溪 | (20, 10-20) | 南北流向，可互动 |
| 萤火虫空地 | (25, 15) | 夜间有萤火虫，浪漫场景 |

---

### 3️⃣ 废弃营地（危险区）

| 属性 | 数值 |
|------|------|
| 位置 | 东侧 |
| 尺寸 | 10×10 瓦片 |
| 植被 | 稀疏，有帐篷残骸 |
| 敌人 | 魔王军侦察兵（第 8 日后） |
| 功能 | 剧情关键、宝箱、情报 |

**特征：**
- 废弃的冒险者营地
- 有帐篷残骸、篝火余烬
- 可找到冒险者日记（剧情线索）
- 第 8 日后被魔王军占领

## 🛤️ 道路系统

### 小径设计

| 类型 | 宽度 | 长度 | 说明 |
|------|------|------|------|
| 主小径 | 2 瓦片 | ~20 瓦片 | 入口→深林区 |
| 支路 | 1 瓦片 | ~5 瓦片 | 通往各特征点 |
| 兽径 | 1 瓦片 | ~3 瓦片 | 隐蔽路径（需探索发现） |

### 小径节点

| 节点 | 位置 | 连接 | 说明 |
|------|------|------|------|
| 森林入口 | (20, 0) | 通往村庄 | 与村南门切换点连接 |
| 小溪渡口 | (20, 15) | 小径交叉口 | 有简易木桥 |
| 蘑菇圈路口 | (10, 5) | 支路分叉 | 通往蘑菇圈 |
| 营地入口 | (25, 10) | 支路终点 | 废弃营地入口 |

## 🌳 植被分布

### 树木

| 类型 | 数量 | 位置 | 说明 |
|------|------|------|------|
| 松树 | 30 棵 | 入口区边缘 | 普通树木，可绕行 |
| 橡树 | 20 棵 | 深林区 | 较大，有阴影效果 |
| 古树 | 3 棵 | 古老树林 | 巨大，不可通行，有秘密 |
| 柳树 | 5 棵 | 小溪边 | 垂柳，装饰用 |

### 地面植被

| 类型 | 数量 | 位置 | 可采集 |
|------|------|------|--------|
| 草地 | 覆盖 60% | 全区域 | 否 |
| 蘑菇 | 15 个 | 蘑菇圈 | ✅ 是（发光蘑菇） |
| 草药 | 10 个 | 入口区、小溪边 | ✅ 是（白芷药园需求） |
| 花朵 | 20 个 | 萤火虫空地 | ✅ 是（装饰用） |

## 💧 水体系统

### 小溪

| 属性 | 数值 |
|------|------|
| 位置 | 南北流向，x=20-22 |
| 长度 | ~20 瓦片 |
| 宽度 | 2-3 瓦片 |
| 深度 | 浅（可看到底） |
| 互动 | 可钓鱼、取水 |

### 水体特效

| 特效 | 说明 | 触发条件 |
|------|------|---------|
| 流动动画 | 水波向南流动 | 持续 |
| 反光效果 | 阳光/月光反射 | 白天/夜晚 |
| 涟漪 | 玩家靠近时 | 距离<3 瓦片 |
| 萤火虫 | 夜间水面有萤火虫 | 夜晚 + 萤火虫空地解锁 |

## 👾 敌人分布（第 8 日后）

## 🎨 视觉风格

### 色调

| 区域 | 主色调 | 辅助色 | 说明 |
|------|--------|--------|------|
| 入口区 | 绿色 | 棕色 | 明亮，安全 |
| 深林区 | 深绿色 | 紫色 | 神秘，压抑 |
| 废弃营地 | 灰褐色 | 黑色 | 荒凉，危险 |

### 昼夜变化

| 时间 | 效果 | 说明 |
|------|------|------|
| 白天 | 阳光透过树叶 | 光斑效果 |
| 黄昏 | 橙红色调 | 浪漫氛围 |
| 夜晚 | 深蓝色 + 萤火虫 | 萤火虫空地发光 |
| 雨天 | 灰暗 + 雾气 | 能见度降低 |

## 🔗 与村庄的关联

### 地理连接

- **村南门** ↔ **森林入口**：无缝切换
- **距离**：从村庄中心到森林入口约 15 瓦片（步行~3 秒）
- **视野**：从森林入口可看到村庄轮廓

### 剧情关联

| 剧情点 | 位置 | 说明 |
|--------|------|------|
| 第 1 日 | 入口区 | 老约翰提示"森林里有答案" |
| 第 3 日 | 萤火虫空地 | 可触发浪漫场景（与特定 NPC） |
| 第 5 日 | 古老树林 | 解锁预言碎片（隐藏） |
| 第 8 日 | 森林入口 | 魔王军封锁，无法进入 |
| 第 10 日 | 废弃营地 | 魔王军主力从此涌出 |
