## 🎮 室内交互系统

### 进入/离开机制

```
1. 靠近建筑门<1 瓦片
   ↓
2. 显示进入提示（按 E/点击）
   ↓
3. 播放进入动画（门打开 + 淡入）
   ↓
4. 加载室内场景
   ↓
5. 放置玩家到室内入口点
```

### 室内相机

| 状态 | 相机行为 | 说明 |
|------|---------|------|
| 进入时 | 平滑过渡到室内视角 | 缩放级别调整 |
| 室内移动 | 跟随玩家，限制在室内 | 不超出墙壁 |
| 离开时 | 平滑过渡到室外视角 | 缩放级别恢复 |

### NPC 室内行为

| NPC | 室内行为 | 离开条件 |
|-----|---------|---------|
| 雷叔 | 熔炉/工作台间移动 | 12:00-13:00 广场午休 |
| 老约翰 | 祭坛/圣经台/房间 | 12:00-13:00 广场布道 |
| 大熊 | 吧台/餐桌间移动 | 10:00-10:30 广场采购 |
| 金铃 | 柜台/货架间移动 | 17:00-20:00 酒馆社交 |
| 小安 | 课桌/画板间移动 | 放学后广场玩耍 |
