## 📐 尺寸规范

### 世界比例

- **1 瓦片（Tile）** = 64x64 像素 = 约 2 米 × 2 米
- **暮色村** = 30×25 瓦片 = 约 60 米 × 50 米
- **迷雾森林** = 40×30 瓦片 = 约 80 米 × 60 米
- **玩家移动速度** = 约 5 瓦片/秒（正常行走）

### 区域间距离

| 路线 | 瓦片数 | 步行时间 | 说明 |
|------|--------|----------|------|
| 暮色村中心 → 森林入口 | ~15 瓦片 | ~3 秒 | 游戏内实时移动 |
| 暮色村 → 魔王城 | ~500 瓦片 | 10 日（剧情） | 背景设定，不可实际移动 |
| 暮色村 → 王国腹地 | ~750 瓦片 | 15 日（背景） | 背景设定，不可实际移动 |

## 🔗 与现有设定的关联

### 故事背景呼应

- **"魔王城脚下的最后一个村庄"** — 地理位置准确，村庄确实在魔王城正南方
- **"十日时限"** — 魔王军从北方进攻，与地图布局一致
- **"黑潮纪元"** — 北方黑潮侵蚀区在地图上可视化

### 角色设定呼应

| 角色 | 位置关联 |
|------|----------|
| 陈默 | 村庄边缘小屋，靠近森林（方便逃避/守望） |
| 雷叔 | 铁匠铺，村庄西侧（传统布局） |
| 金铃 | 商人行会，村庄东侧（交通便利） |
| 白芷 | 药园，村庄边缘（需要安静） |
| 老约翰 | 教堂/广场，村庄中心（精神核心） |
| 大熊 | 酒馆，村庄中心（信息中心） |
| 影 | 独居，村庄边缘（隐蔽） |
| 小安 | 学校，村庄中心（被保护） |
| 阿虎 | 守卫哨站/营房，村庄入口（职责所在） |
| 夜鸦 | 观察点，可俯瞰全村（监视任务） |
