```

   ↓
3. 播放采集动画
   ↓
4. 获得物品（数量随机）
   ↓
5. 采集点进入冷却（刷新时间后恢复）
```

## 🎮 交互系统设计

### 交互类型

| 类型 | 输入 | 说明 |
|------|------|------|
| 对话 | 靠近 + 点击/E | 与 NPC 对话 |
| 采集 | 靠近 + 点击/E | 采集草药、蘑菇等 |
| 开启 | 靠近 + 点击/E | 打开容器 |
| 使用 | 靠近 + 点击/E | 使用机关 |
| 阅读 | 靠近 + 点击/E | 阅读信件、日记 |
| 坐下 | 靠近 + 点击/E | 坐在长椅上 |
| 调查 | 靠近 + 点击/E | 调查物体 |

### 交互距离

| 交互类型 | 触发距离 | 说明 |
|---------|---------|------|
| NPC 对话 | <2 瓦片 | 显示对话气泡 |
| 采集 | <1 瓦片 | 伸手可及 |
| 容器 | <1.5 瓦片 | 靠近容器 |
| 机关 | <2 瓦片 | 可操作范围 |

### 交互反馈

| 反馈类型 | 效果 | 说明 |
|---------|------|------|
| 视觉 | 高亮显示 | 鼠标悬停时物体高亮 |
| 听觉 | 音效 | 不同交互有不同音效 |
| 文本 | 提示文字 | 显示"按 E 交互" |
| 动画 | 交互动画 | 采集、开启等动画 |

## 🔗 与相关系统的关联

### 信任系统

- NPC 交互提升/降低信任值
- 信任值影响对话选项和容器开启权限

### 采集系统

- 采集物有刷新时间
- 部分采集需要 NPC 许可

### 事件触发系统

- 剧情物体触发关键事件
- 机关交互影响游戏状态

### 存档系统

- 容器开启状态需要保存
- 采集点刷新时间需要保存
