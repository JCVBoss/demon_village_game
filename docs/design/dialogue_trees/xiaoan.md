# 🎭 小安（Xiao An）完整对话树

> "叔叔说爸爸去了很远的地方...他会回来吗？"

---

## 📋 角色信息

| 项目 | 设定 |
|------|------|
| **NPC ID** | `xiaoan` |
| **表面身份** | 村里最可爱的孩子，8 岁 |
| **真实身份** | 陈默队长的亲生儿子 |
| **信任阶段** | 3 个阶段（陌生→熟悉→依赖） |
| **核心秘密** | 不知道陈默的真实身份（教父），以为爸爸"去了很远的地方" |
| **功能定位** | 情感核心、隐藏任务、触发陈默成长线 |
| **对话风格** | 天真、聪明、敏感、能感觉到大人的情绪 |

---

## 🌳 对话树总览

```
阶段 1：陌生（信任 0-30）
├─ 初次见面 → 好奇问候
├─ 陪孩子玩 → 开心分支
└─ 询问爸爸 → 天真回答分支

阶段 2：熟悉（信任 31-60）
├─ 再次见面 → 主动打招呼
├─ 询问爸爸详情 → 怀疑分支
└─ 赠送礼物 → 感激分支

阶段 3：依赖（信任 61-100）
├─ 深入对话 → 情感分支
├─ 询问真相 → 关键抉择 🔓
└─ 承诺保护 → 信任分支

特殊节点
├─ 第 5 日村庄广场 → 孩子提问
├─ 第 8 日身世揭露 → 关键剧情 🔓
└─ 第 10 日决战后 → 结局分支
```
