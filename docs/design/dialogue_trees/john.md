## 🌳 对话树总览

```
阶段 1：陌生（信任 0-20）
├─ 初次见面 → 预言碎片 1 解锁
├─ 询问教堂 → 基础信息
└─ 祈祷 → 信任 +3

阶段 2：好奇（信任 21-40）
├─ 询问预言 → 碎片 2 线索
├─ 询问过去 → 回避/暗示
└─ 请求净化 → 精神恢复

阶段 3：信赖（信任 41-60）
├─ 询问勇者真相 → 碎片 3 解锁
├─ 质疑预言 → 考验分支
└─ 深入祈祷 → 碎片 4 线索

阶段 4：托付（信任 61-100）
├─ 坦白知道真相 → 碎片 5 解锁 🔓
├─ 询问最终选择 → 结局引导
└─ 揭露身份（真结局线）→ 真相结局前置 🔓

特殊节点
├─ 第 5 日专属 → 预言仪式
├─ 第 9 日专属 → 最终抉择前
└─ 第 10 日专属 → 结局分支
```
