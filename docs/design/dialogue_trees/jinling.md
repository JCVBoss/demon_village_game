---

## 📋 角色信息

| 项目 | 设定 |
|------|------|
| **NPC ID** | `jinling` |
| **表面身份** | 往来于各村落的行商 |
| **真实身份** | 前盗贼团"夜鸦"二把手，金盆洗手 |
| **信任阶段** | 3 个阶段（警惕→交易→信任） |
| **核心秘密** | 知道观察者（夜鸦）的真实身份 |
| **功能定位** | 道具交易、情报贩卖、特殊任务触发 |
| **对话风格** | 精明算计、爱财但守底线、擅长察言观色 |

## 🌳 对话树总览

```
阶段 1：警惕（信任 0-30）
├─ 初次见面 → 警惕询问
├─ 询问商品 → 交易分支
└─ 询问情报 → 条件分支

阶段 2：交易（信任 31-60）
├─ 再次拜访 → 交易分支
├─ 完成任务 → 信任分支
└─ 询问过去 → 回避分支

阶段 3：信任（信任 61-100）
├─ 深入对话 → 信任分支 🔓
├─ 询问夜鸦 → 秘密分支 🔓
└─ 请求帮助 → 全力分支

特殊节点
├─ 第 2 日商人到来 → 初次交易
├─ 第 6 日商人情报 → 找内鬼任务
└─ 第 10 日决战前 → 物资支援
```
