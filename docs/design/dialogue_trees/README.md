# 🎭 对话树设计规范 v2.0

> 一周目预设对话树设计标准

---

## 📋 设计原则

### 双模式架构

| 模式 | 适用场景 | 对话类型 |
|------|----------|----------|
| **一周目** | 首次游玩 | 预设分支对话（本规范） |
| **二周目** | 通关后解锁 | LLM 动态生成（见人格卡） |

### 核心原则

1. **关键剧情固定** — 秘密解锁、最终章等核心剧情使用固定文本
2. **日常对话可分支** — 日常互动提供 2-4 个分支选项
3. **信任值驱动** — 对话内容随信任值动态解锁
4. **信息传播** — 重要事件通过事件标志在其他 NPC 对话中体现

## 📊 对话量标准

### 按信任阶段分配

| 阶段 | 信任范围 | 场景数 | 分支数 | 对话条数 |
|------|----------|--------|--------|----------|
| 阶段 1 | 0-20 | 3-4 | 2-3 | ~10 |
| 阶段 2 | 21-40 | 3-4 | 2-3 | ~12 |
| 阶段 3 | 41-60 | 3-4 | 2-3 | ~12 |
| 阶段 4 | 61-100 | 4-5 | 2-3 | ~15 |
| 特殊 | - | 3-5 | 1-2 | ~10 |
| **总计** | - | **16-20** | **~2.5** | **~59** |

### 按游戏日程分配

| 时期 | 天数 | 对话重点 |
|------|------|----------|
| 初期 | 第 1-3 日 | 初识、基础信息 |
| 中期 | 第 4-6 日 | 关系发展、秘密线索 |
| 后期 | 第 7-9 日 | 秘密揭露、情感高潮 |
| 终章 | 第 10 日 | 最终抉择、结局分支 |

## 🎯 对话设计要点

### 1. 信任值变化设计

**信任值获取方式：**

| 行为 | 信任变化 | 说明 |
|------|----------|------|
| 友好对话 | +3~5 | 日常友好互动 |
| 帮助行为 | +5~10 | 完成 NPC 请求 |
| 赠送礼物 | +5~15 | 根据喜好 |
| 提及禁忌 | -5~-10 | 触及伤痛/秘密 |
| 背叛行为 | -20~-50 | 严重背叛 |

### 2. 分支选项设计

**选项类型：**

```json
{
  "choice_types": {
    "friendly": "友好回应，稳定 + 信任",
    "curious": "好奇询问，可能 + 信任或 - 信任",
    "confrontational": "对抗/质疑，高风险高回报",
    "helpful": "提供帮助，通常 + 信任",
    "avoidant": "回避/离开，信任不变或微降"
  }
}
```

### 3. 秘密解锁设计

**秘密层级：**

| 层级 | 信任要求 | 内容 | 影响 |
|------|----------|------|------|
| L1 | 30-40 | 表面秘密 | 解锁新对话 |
| L2 | 50-60 | 个人背景 | 解锁支线任务 |
| L3 | 70-80 | 核心秘密 | 解锁结局分支 |
| L4 | 90-100 | 完全坦白 | 专属结局 |

## 🔗 信息传播机制

### 事件标志系统

```json
{
  "event_flags": {
    "chenmo_secret_revealed": {
      "description": "陈默的秘密被揭露",
      "trigger": "玩家与陈默信任≥80 并解锁秘密",
      "propagation": [
        {
          "npc": "da_xiong",
          "delay_days": 1,
          "trust_effect": 10,
          "dialogue_node": "daxiong_chenmo_secret"
        },
        {
          "npc": "bai_zhi",
          "delay_days": 1,
          "trust_effect": 5,
          "dialogue_node": "baizhi_chenmo_past"
        }
      ]
    }
  }
}
```

### 传播规则

1. **关键事件**（秘密揭露、重大选择）→ 1 天后传播
2. **日常事件**（帮助、对话）→ 不传播或轻微传播
3. **负面事件**（背叛、冲突）→ 传播速度×2
