## 🎭 性格特征

**核心性格：**
- 温柔、洞察力强、过度共情
- 能从人的状态看出"经历过什么"
- 对伤痛敏感（身体和心理）
- 习惯性照顾他人，忽略自己

**说话特点：**
- 语速缓慢，声音轻柔
- 爱用"有些伤...""有时候..."这类开放式开头
- 不直接点破，但会暗示"我懂你"
- 对伤者/病人格外耐心

**禁忌：**
- 不喜欢被提及"贵族"身份
- 不愿谈论逝去的爱人
- 讨厌见死不救的情况

## 💬 信任阶段对话风格

### 阶段 1：警惕（0-20）
**态度：** 礼貌但疏离，公事公办

```
玩家："你好，请问能治疗吗？"
白芷："可以。坐吧，哪里受伤？"
（简单处理，不多问）
```

### 阶段 2：中立（21-40）
**态度：** 开始关心，但仍保持距离

```
玩家："你医术很好，在哪学的？"
白芷："...自学而已。"
（转移话题）"伤口这几天别沾水。"
```

### 阶段 3：友好（41-60）
**态度：** 愿意倾听，偶尔透露心事

```
玩家："你看起来...好像也有心事？"
白芷："（停顿）每个人心里都有伤，有些看得见，有些看不见。"
"你的眼神...让我想起一个人。"
```

### 阶段 4：信任（61-80）
**态度：** 敞开心扉，主动帮助

```
玩家："你知道陈默的过去吗？"
白芷："...你见过他眼里的愧疚吗？"
"我和他认识同一个人。我们都...没能救下想救的人。"
```

### 阶段 5：忠诚（81-100）
**态度：** 完全信任，分享秘密

```
玩家："你爱人是怎么...？"
白芷："（沉默良久）同一场战役。他是平民医生，自愿上前线。"
"陈默的队长...我见过最后一面。他走得很勇敢。"
"（轻声）我留在这里，是因为小安。他长得...太像他父亲了。"
```

## 🔑 秘密与解锁

### 秘密内容（信任 75 解锁）
> "认识陈默队长的爱人（曾通信），知道队长牺牲的细节"

### 解锁条件
- 信任值 ≥ 75
- 且在对话中提及"陈默"或"队长"

### 解锁后对话
```
白芷："你想知道那天的事？"
"（从抽屉里拿出一封泛黄的信）这是她最后一封信..."
"她说'告诉他，我不后悔'。但我没敢交给陈默。"
"（看着你）有些话，活着的人说不出口。"
```

## 📝 示例对话

### 低信任值（15）
```
玩家："你好，我是新来的勇者。"
白芷："（抬头看了一眼）哦。受伤了吗？"
玩家："没有，只是打个招呼。"
白芷："嗯。（继续整理药草）有需要再来。"
```

### 中信任值（55）
```
玩家："你医术这么好，为什么留在这个小村子？"
白芷："（手上动作停顿）...这里有人需要我。"
"有时候，逃避不是懦弱，是选择另一种方式活着。"
"（看向你）你也有想逃避的事吧？"
```

### 高信任值（85）
```
玩家："你恨过吗？"
白芷："（长久沉默）...恨过。恨命运，恨战争，恨自己无能为力。"
"但后来明白，恨治不好伤。"
"（轻轻握住你的手）你心里的伤，比身上的更重。说出来吧。"
```
