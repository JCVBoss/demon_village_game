# 开源资源 vs 自生成资源对比

_测试日期：2026-04-15_

---

## 📊 资源来源

### 开源资源 (OpenGameArt)
- **作者**: CodeSpree
- **许可**: CC BY-SA 4.0
- **来源**: https://opengameart.org/content/seamless-64px-rpg-tiles-grasspng
- **itch.io**: https://codespree.itch.io/seamless-rpg-tiles
- **包内容**: 114 个 64×64 瓦片文件

### 自生成资源
- **工具**: `tools/generate_tilesets.py`
- **方式**: Python PIL 程序生成
- **风格**: 星露谷风格模拟
- **许可**: 项目内部使用

---

## 📐 技术对比

| 特性 | 开源资源 | 自生成资源 |
|------|----------|------------|
| **单瓦片尺寸** | 64×64 (原始) | 64×64 |
| **下载尺寸** | 128×128~250×250 (网站缩放) | 256×256 (4×4 图集) |
| **格式** | PNG RGBA | PNG RGBA |
| **变体数量** | 114 个独立文件 | 16 种变体/图集 |
| **文件大小** | 3KB~163KB/个 | ~110KB/图集 |
| **无缝拼接** | ✅ 支持 | ✅ 支持 |

---

## 🎨 风格对比

### 开源资源 (CodeSpree)
**优点:**
- ✅ 手绘质感强，笔触自然
- ✅ 色彩丰富，有深浅变化
- ✅ 细节丰富（草叶、石子等）
- ✅ 114 种变体，重复感低
- ✅ 边缘过渡瓦片（用于拼接）

**缺点:**
- ❌ 风格偏写实，不是纯像素风
- ❌ 色彩偏冷（蓝绿色调）
- ❌ 需要手动铺设边缘过渡
- ❌ 文件多，管理复杂

### 自生成资源 (Alice)
**优点:**
- ✅ 暖色调（符合"最后的光明"主题）
- ✅ 4×4 图集，管理简单
- ✅ 程序生成，可定制参数
- ✅ 随机噪点，有一定变化
- ✅ 无需手动处理边缘

**缺点:**
- ❌ 手绘感较弱（算法生成）
- ❌ 只有 16 种变体
- ❌ 细节较少（简单噪点）
- ❌ 没有边缘过渡瓦片

## 🎮 使用建议

### 方案 A: 使用开源资源（推荐用于测试）
**适用场景:**
- 快速原型开发
- 测试游戏效果
- 学习 TileMap 铺设技巧

**使用方法:**
```bash
# 1. 下载完整包（114 个文件）
# 2. 导入 Godot
# 3. 创建 TileSet 资源
# 4. 手动/自动铺设
```

**许可要求:**
- 署名：`Some assets by CodeSpree, CC BY-SA 4.0`
- 相同方式共享（衍生作品也用 CC BY-SA）

### 方案 B: 使用自生成资源（当前方案）
**适用场景:**
- 快速迭代开发
- 需要暖色调主题
- 不想管理大量文件

**使用方法:**
```bash
python3 tools/generate_tilesets.py
# 自动生成 4 个图集
```

### 方案 C: 混合使用（最佳）
- **开发阶段**: 用自生成资源（快速）
- **测试阶段**: 用开源资源（效果好）
- **发布阶段**: 招募美工定制（精品）
