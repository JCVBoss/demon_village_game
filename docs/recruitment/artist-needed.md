## 📋 项目简介

《暮色村》是一款星露谷物语风格的模拟经营游戏，玩家扮演村长管理村庄、与村民互动、解锁秘密。

- **引擎**: Godot 4.x
- **视角**: 2D 斜角俯视（伪 3D）
- **渲染**: Y-sorting 自动遮挡
- **主题**: "最后的光明" - 温暖、希望、中世纪奇幻

**GitHub**: https://github.com/JCVBoss/demon_village_game

## 📦 交付格式

### 技术规范
- **格式**: PNG（透明背景）
- **色彩**: 32 位 RGBA
- **基准单位**: 64×64 像素
- **图集**: 按类型打包（如 grass.png 256×256）

## 🙋 常见问题

**Q: 需要会 Godot 吗？**  
A: 不需要，只需提供 PNG 资源，代码团队负责集成。

**Q: 必须完全星露谷风格吗？**  
A: 参考星露谷，但鼓励个人风格，符合"最后的光明"主题即可。

**Q: 可以只接 AR-023 或 AR-024 吗？**  
A: 可以，欢迎分项合作。

**Q: 修改次数有限制吗？**  
A: 合理修改（3 次内）包含在报价内，大幅返工需协商。
