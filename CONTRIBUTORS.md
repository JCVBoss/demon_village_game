# 项目说明

**魔王城下的最后村庄** 是一个使用 Godot 4.6 开发的叙事驱动像素风 RPG。

## 团队

| 角色 | 成员 | 职责 |
|------|------|------|
| 项目所有者 | JCVBoss | 整体规划、决策、审核 |
| 开发 | Alice (AI Agent) | 代码开发、文档编写、资源整合 |

## 工作目录

| 目录 | 说明 |
|------|------|
| `code/` | Godot 项目源码（场景、脚本、资源、数据） |
| `docs/design/` | 游戏设计文档（角色、故事、系统、对话树） |
| `tools/` | 资源生成辅助工具 |
| `daily_logs/` | 开发日志 |

## 提交规范

```
<type>(<scope>): <subject>
```

- `feat` - 新功能 | `fix` - 修复 | `docs` - 文档 | `refactor` - 重构 | `art` - 美术
