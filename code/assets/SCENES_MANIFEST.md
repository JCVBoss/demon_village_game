## 🎨 设计理念

### 色彩主题
- **白天:** 明亮清新，蓝天绿地
- **黄昏:** 紫橙渐变，温暖氛围（主视觉）
- **夜晚:** 深蓝调，房屋透光，星空

### 像素风格
- 保持与角色立绘一致的像素精度
- 色彩层次丰富（3 阶渐变）
- 符合中世纪奇幻主题

## 🔧 生成工具

**工具位置:** `tools/scene_generator.py`

**使用方法:**
```bash
python3 tools/scene_generator.py
```

**可定制:**
- 调整色板（SCENE_COLORS）
- 修改建筑位置（house_positions）
- 修改树木位置（tree_positions）
- 添加更多场景元素
